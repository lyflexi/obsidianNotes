# 依赖关系

不同于关联关系，依赖不具有“拥有关系”，而是一种“相识关系”，只在某个特定地方持有==局部变量引用==（比如某个方法体内）才有关系。
![[Pasted image 20231225110337.png]]

下面都属于使用依赖的具体形式：
- 使用（《use》）：方法局部变量引用
- 调用（《call》）：调用服务方的静态方法
- 参数（《parameter》）：持有方法参数引用 
- 返回值（《result》）：持有方法返回值
- 发送（《send》）：表征数据发出的流向
- 实例化（《instantiate》）
